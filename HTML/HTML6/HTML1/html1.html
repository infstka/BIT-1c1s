HTML

Краткое введение

HTML (Hyper Text Markup Language) – язык разметки гипертекста.
Для разметки HTML документа используют теги (флаги разметки).
Тег – это определенная последовательность символов, заключенные между знаками больше и меньше.

Таблицы в HTML

Элемент (table) служит контейнером для элементов, определяющих содержимое таблицы.
Любая таблица состоит из строк и ячеек, которые задаются с помощью тегов (tr) и (td).
Для отображения границ таблицы используется атрибут border.

Ссылки на Web-страницах

Ссылки являются основой гипертекстовых документов и позволяют переходить с одной веб-страницы на другую.
Реализуются с помощью тега A.
Атрибут href определяет URL адрес документа, на который следует перейти (например "cat.html"), а содержимое контейнера (a) являетсяссылкой.
